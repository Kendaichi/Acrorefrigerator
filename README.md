# Acro Refrigeration

Commercial refrigeration contractor website for Acro Refrigeration, Sydney.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev

# Build for production
npm run build
```

## Testing

```sh
npm test
```
